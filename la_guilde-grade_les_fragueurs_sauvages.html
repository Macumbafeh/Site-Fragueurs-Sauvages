Les grades de la guilde Les Fragueurs Sauvages







Les grades par ordre croissant :


  • Eclaireur / Eclaireuse : Nouveau arrivant et joueur(euse) qui n'a pas de stuff (Full vert/bleu) 13 points c'est le minimum.

  • Sergent / Sergente : Joueur avec au minimum le trinket PvP et du 2 pièces s0. 25 points requis.

  • Adjudant / Adjudante : Joueur(euse) avec armes s2 et une pièce s4 (set ou hors set). 32 points requis.

  • Garde de pierre : Joueur(euse) avec une pièce de set s4, au moins une s2, et deux hors set s4 (ou inversement). 40 points requis.

  • Centurion : Joueur(euse) en cours de stuff avec 1 pièce s4 et deux pièces s2 et trois hors set s4. 46 points requis.

  • Champion / Championne : Joueur(euse) Bientôt stuff avec 2 pièce s4 reste s2 et 3 hors set s4 avec bague s3. 52 points requis.

  • Lieutenant / Lieutenante : Joueur(euse) stuff au minimum 4/5 s4 avec hors set s4 (voir un hors set s3) arme s2 et épaules s2. 59 points requis.

  • Général / Générale : Grade d'Officier, qui seconde le Maitre de guilde.

  • Grand seigneur : Grade de Maitre de Guilde qui gère la guilde.

Exception : Les armes et stuff pve HL sont compté comme du stuff, comme les armes forgerons, les casques ingé, ect...
Les bijoux à part le trinket PvP, ne sont pas compté et la cape aussi.


Points par stuff :


Set : 5 pièces

Stuff vert / bleu set et hors set = 1 points (à part pve HL = 2 points)
Set s0 = 2 points
Set s2 = 3 points
Set s3 = 4 points
Set s4 = 5 points


Hors set : 6 pièces

Hors set vert / bleu = 1 points (à part pve HL = 2 points)
Hors set s3 = 2 points
Hors set s4 = 4 points


Armes une main et tenu en main gauche : 1 à 2 pièces

Armes vert / bleu = 1 points (pve HL = 2 voir 3 points selon le niveau)
Armes s2 = 3 points
Armes s4 = 4 points


Armes deux mains : 1 pièce

Armes vert / bleu = 2 points (Pve HL et armes métier 4 voir 6 points selon le niveau)
Armes s2 = 6 points
Armes s4 = 8 points

Trinket PvP = 10 points



Le calculs du stuff :


Un full s4 aura 65 points. (bague s3 et trinket pvp)

Un Unstuff full vert/bleu (niveau le plus bas) aura 13 points (pas de trinket pvp).

Un unstuff avec le trinket PvP aura 23 points.

Full s0 avec trinket aura 28 points.

un Full s2 avec gants s4 et trinket pvp, sans hors set, aura 32 points.

Celui qui est full Pve HL (genre t6) aura 28 points.

ect...,ect...
